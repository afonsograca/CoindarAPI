A wrapper around the [Coindar API](https://coindar.org/en/api) written in Swift

## Installation

CoindarAPI is available through [CocoaPods](https://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod 'CoindarAPI'
```
